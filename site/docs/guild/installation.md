# 安装

## 环境支持

由于 Vue 3 不再支持 IE11，neat-design 也不再支持 IE11 浏览器。

## 版本

neat-design 目前还在开发中

## 使用包管理器

建议您使用包管理器 (NPM, Yarn, pnpm) 安装 neat-design, 然后您就可以使用打包工具，例如 Vite 和 webpack

```
# 选择一个你喜欢的包管理器

# NPM
$ npm install neat-design --save

# Yarn
$ yarn add neat-design

# pnpm
$ pnpm install neat-design
```

## 浏览器直接引入
